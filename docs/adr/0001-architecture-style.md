# ADR 0001: Architecture Style

## Контекст
Проєкт розробляється одним студентом.
Команда маленька, процеси прості.
Система — навчальний каталог товарів.

## Рішення
Обрано архітектурний стиль: модульний моноліт.

## Альтернативи
Мікросервіси були розглянуті, але відхилені через складність
та відсутність реальної потреби на цьому етапі.

## Наслідки
- Простий запуск і розробка
- Єдина кодова база
- Можливість перейти до мікросервісів у майбутньому

## Статус
Прийнято

## Дата
2025
