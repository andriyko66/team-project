# Каталог товарів

## Опис системи
Система дозволяє користувачам переглядати каталог товарів. 
Користувач натискає кнопку і бачить усі товари з бази. 
У майбутньому можна додати авторизацію та сповіщення.

## Ролі користувачів
- **Користувач** — переглядає товари.
- **Адміністратор** — додає або редагує товари (майбутнє).

## Основні сценарії
1. Користувач відкриває сторінку.
2. Натискає "Load items".
3. Система показує список товарів.
4. Адміністратор додає новий товар.
5. Система оновлює список.

## Функціональні вимоги
- Показувати товари з бази.
- Додавати тестові товари через бекенд.

## Нефункціональні вимоги
- Швидке завантаження (<2 сек)
- Зручний інтерфейс

## Обмеження
- Не редагуємо та не видаляємо товари.
- Не робимо оплату або чат.
