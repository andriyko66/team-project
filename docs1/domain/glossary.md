# Глосарій термінів (Ubiquitous Language)

**Product (Товар)**  
Окрема одиниця каталогу, яка має назву, ціну та категорію.

**Category (Категорія)**  
Група товарів, яка використовується для структурування каталогу.

**Catalog (Каталог)**  
Набір усіх товарів і категорій, доступних для перегляду.

**User (Користувач)**  
Людина, яка користується системою для перегляду товарів.

**Price (Ціна)**  
Грошова вартість товару.

**Product ID**  
Унікальний ідентифікатор товару в системі.
